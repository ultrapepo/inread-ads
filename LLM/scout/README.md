# Scout Output (Optional)

If you run the optional Scout Initialization step (see `LLM/SCOUT_PROMPT.md`), the Orchestrator should create:

- `LLM/scout/baseline.md` — a small baseline “project skeleton” containing:
  - **Verification Anchors:** how to build/run/test
  - **Navigation Anchors:** where key entry points and conventions live

This file is intentionally **small** and **incomplete**. It exists to make the first few handoffs accurate without trying to map every feature up front.
