# Handoffs

This folder contains Coding LLM handoff specs written by the Orchestrator.

Rules of thumb:
- One handoff per feature/task: `LLM/handoffs/{feature}.md`
- Update in place (avoid `*_V2.md`, `*_FINAL.md`)
- `LLM/CURRENT_TASKS.md` should point at the active handoff path
- Keep old handoffs: they are useful when resuming work or when the user references a previous feature
